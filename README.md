# ATM-Console-Application
This is a console-based ATM application developed to simulate basic ATM operations.

## Features
- User authentication using PIN
- Balance inquiry
- Cash withdrawal
- Deposit amount
- Exit option

## Technologies Used
- Python 

## How to Run
1. Download or clone the repository
2. Open terminal / command prompt
3. Run the main file

## Learning Outcome
- Improved logical thinking
- Better understanding of conditions and loops
- Real-world problem simulation
